# Estudio Jurídico - Sitio web minimalista

Proyecto Next.js minimalista listo para desplegar en Vercel.

## Instrucciones rápidas para desplegar en Vercel

1. Crear un repositorio nuevo en GitHub.
2. Subir esta carpeta (todos los archivos y carpetas) al repositorio (puedes arrastrar los archivos en la opción 'Upload files').
3. Crear cuenta en https://vercel.com y conectar tu cuenta de GitHub.
4. En Vercel, seleccionar **Import Project** → elegir el repositorio subido → Deploy.
5. Reemplazar `/public/logo.png` por tu logo real (misma resolución o similar).
6. Editar `pages/index.js` para actualizar número de WhatsApp o textos que desees cambiar.

## Estructura del proyecto
- /public/logo.png  -> logo de prueba (blanco y negro)
- /pages            -> archivos de páginas (index.js)
- /components/ui    -> componentes simples (Card, Button)
- /styles/globals.css -> estilos globales (Tailwind compatibles)

Si necesitas que suba esto a un repositorio GitHub por ti, puedo guiarte paso a paso o ayudarte a crear commits desde tu máquina.
